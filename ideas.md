# Selected Design Philosophy

I will proceed with the **Modern Sports Analytics** approach. This style aligns best with the "Mobile Coach" and "Develops" branding of USA Baseball, which emphasizes professional development and structured training. It provides a clean, organized interface for navigating a large dataset (282 drills) while maintaining a professional, athletic aesthetic.

**Key Design Decisions:**
- **Color Palette**: Navy Blue (#002D62), Red (#BF0D3E), White, Cool Grey.
- **Typography**: 'Oswald' for headings (athletic feel), 'Inter' for UI text (readability).
- **Layout**: Responsive grid of drill cards with a sticky search/filter header.
- **Visuals**: Clean, card-based UI with clear category indicators.
